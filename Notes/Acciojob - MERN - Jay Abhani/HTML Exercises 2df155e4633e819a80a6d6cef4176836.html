HTML Exercises

1️⃣ Create a To-Do List for Travelers

  • Create an unordered list (<ul>) of essential travel items (e.g., Passport, Clothes, Chargers, Camera).
  • Add a heading ("Packing Checklist") above the list.

2️⃣ Design a Navigation Menu for a Hotel Website

  • Use an unordered list (<ul>) inside a <nav> element.
  • Include links to Home, Rooms, Facilities, Reviews, Contact Us.

3️⃣ Display a List of Amenities in an Airbnb Property Page

  • Use a description list (<dl>) where:
    • <dt> contains "WiFi", "Swimming Pool", "24/7 Customer Support".
    • <dd> explains each amenity briefly.

4️⃣ Create a Food Menu for a Restaurant Page

  • Use an ordered list (<ol>) for "Starters, Main Course, Desserts".
  • Add list items (<li>) for dishes under each category.

5️⃣ Build a Hotel Booking Process Step List

  • Use an ordered list (<ol>) to explain:
    • Step 1: Search for a Hotel
    • Step 2: Select a Room
    • Step 3: Enter Guest Details
    • Step 4: Make Payment

Convert a Non-Semantic Webpage into Semantic HTML

  • Take a webpage built with only <div> and <span>.
  • Replace them with <header>, <nav>, <section>, <article>, and <footer> for better structure.

🔴 Before: Non-Semantic HTML (Using <div> and <span> Everywhere)

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Non-Semantic Page</title>
</head>
<body>
    <div>
        <div>
            <span>My Website</span>
        </div>
        <div>
            <a href="#">Home</a>
            <a href="#">About</a>
            <a href="#">Services</a>
            <a href="#">Contact</a>
        </div>
    </div>

    <div>
        <div>
            <h2>Welcome to My Website</h2>
            <p>This is a basic webpage structure using only divs and spans.</p>
        </div>
        <div>
            <h3>Our Services</h3>
            <p>We provide web development and design solutions.</p>
        </div>
    </div>

    <div>
        <p>&copy; 2025 My Website. All rights reserved.</p>
    </div>
</body>
</html>
