Nested Arrays

🔹 What is an Array?

An array is a special variable that can hold multiple values in a single variable.

let fruits = ["apple", "banana", "orange"];

Here, fruits is an array that holds 3 string values.

🔸 What is a Nested Array?

A nested array is an array that contains other arrays inside it.

let numbers = [[1, 2], [3, 4], [5, 6]];

In this example, numbers is a nested array with 3 inner arrays:

  • [1, 2]
  • [3, 4]
  • [5, 6]

Each of these inner arrays is an element of the main array.

🔹 Accessing Values in Nested Arrays

To access elements inside a nested array, use two index positions:

let numbers = [[1, 2], [3, 4], [5, 6]];

console.log(numbers[0]);     // [1, 2] — the first array
console.log(numbers[0][0]);  // 1      — first element of first array
console.log(numbers[1][1]);  // 4      — second element of second array

🔸 Real-Life Example

Imagine a classroom with rows of students. Each row is an array, and the whole classroom is a nested array.

let classroom = [
  ["Jay", "Neha", "Raj"],    // Row 1
  ["Ankit", "Pooja", "Simran"], // Row 2
  ["Amit", "Riya", "Tina"]   // Row 3
];

console.log(classroom[1][2]); // Simran — 3rd student of 2nd row

🔹 Looping Through Nested Arrays

You can use nested loops to go through each element.

let numbers = [[1, 2], [3, 4], [5, 6]];

for (let i = 0; i < numbers.length; i++) {
  for (let j = 0; j < numbers[i].length; j++) {
    console.log(numbers[i][j]);
  }
}

This prints:

1
2
3
4
5
6

🔸 Adding/Modifying Elements in Nested Arrays

Add a new inner array:

numbers.push([7, 8]);
console.log(numbers); // [[1,2], [3,4], [5,6], [7,8]]

Change a specific value:

numbers[0][1] = 99;
console.log(numbers); // [[1,99], [3,4], [5,6]]

🔹 Why Use Nested Arrays?

You use them when you need to organize data in rows/columns or in groups. Example:

  • Tables
  • Grids (like in games)
  • Matrix operations
  • Storing grouped data
