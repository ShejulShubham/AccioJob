SelectorBest For
* (Universal)Reset styles globally
elementStyling all occurrences of an element
.classReusable styles for multiple elements

div pTargeting nested elements

  1. Descendant Selector (A B)
    • Selects all elements B that are inside element A (at any depth).
    • Example: div p → selects all <p> inside <div>.
  1. Child Selector (A > B)
    • Selects elements B that are direct children of A (not grandchildren).
    • Example: div > p → selects only immediate child <p> inside <div>.
  1. Adjacent Sibling Selector (A + B)
    • Selects the immediately next sibling element B that comes right after A.
    • Example: h2 + p → selects the first <p> immediately after an <h2>.
  1. General Sibling Selector (A ~ B)
    • Selects all siblings B that come after A (not just the first).
    • Example: h2 ~ p → selects all <p> that come after an <h2>.

    h2 + p {
        color: green;
    }
    

    /* 4. General Sibling Selector */
    /* All p after h2 (siblings) */
    h2 ~ p {
        background-color: gray;
    }

    /* 5. Attribute Selector */
    /* All h3 having attribute class and its value "ethics" */
    h3[class="ethics"] {
        color: brown;
    }
