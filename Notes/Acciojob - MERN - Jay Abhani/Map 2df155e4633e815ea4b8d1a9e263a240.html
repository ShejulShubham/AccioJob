📘 Map in JavaScript

🔹 What is a Map?

In JavaScript, a Map is a built-in object that holds key-value pairs and remembers the original insertion order of the keys.

Unlike regular JavaScript objects:

  • Any value (object or primitive) can be used as a key.
  • Keys are ordered.
  • Map provides a set of built-in methods for easy manipulation.

🔹 Creating a Map

✅ Using Constructor

let myMap = new Map();

✅ With Initial Values

let myMap = new Map([
  ["name", "Alice"],
  ["age", 25],
  ["isStudent", true]
]);

🔹 Common Map Methods

1. set(key, value)

Adds or updates an entry.

myMap.set("language", "JavaScript");

2. get(key)

Retrieves a value by key.

myMap.get("language"); // "JavaScript"

3. has(key)

Checks if the map contains a key.

myMap.has("language"); // true

4. delete(key)

Removes a key-value pair.

myMap.delete("language");

5. clear()

Removes all entries from the map.

myMap.clear();

6. size Property

Returns the number of entries.

myMap.size;

🔹 Iterating over a Map

for...of loop:

for (let [key, value] of myMap) {
  console.log(`${key}: ${value}`);
}

forEach() method:

myMap.forEach((value, key) => {
  console.log(`${key} => ${value}`);
});

keys(), values(), and entries()

for (let key of myMap.keys()) {
  console.log(key);
}

for (let value of myMap.values()) {
  console.log(value);
}

for (let [key, value] of myMap.entries()) {
  console.log(`${key}: ${value}`);
}

🔹 Using Objects or Functions as Keys

let objKey = { id: 1 };
let funcKey = function () {};

let myMap = new Map();

myMap.set(objKey, "Object as key");
myMap.set(funcKey, "Function as key");

console.log(myMap.get(objKey)); // Object as key
console.log(myMap.get(funcKey)); // Function as key
🧠 Unlike regular objects, Map preserves reference identity of object keys.

🔹 Convert Map to Array

let arr = [...myMap];
// or
Array.from(myMap);

Convert Map to Object (if keys are strings):

let obj = Object.fromEntries(myMap);

🔹 Convert Object to Map

let obj = { a: 1, b: 2 };
let map = new Map(Object.entries(obj));

🔹 Real-World Use Cases of Map

  1. Storing configuration: Dynamic keys and values
  1. Caching data: Using functions or objects as keys
  1. Counting occurrences: Frequency maps
  1. Maintaining insertion order: Unlike plain objects

✅ Example: Frequency Counter Using Map

function countWords(text) {
  let words = text.split(" ");
  let freqMap = new Map();

  for (let word of words) {
    word = word.toLowerCase();
    freqMap.set(word, (freqMap.get(word) || 0) + 1);
  }

  return freqMap;
}

let result = countWords("hello world hello");
console.log(result); // Map { 'hello' => 2, 'world' => 1 }
