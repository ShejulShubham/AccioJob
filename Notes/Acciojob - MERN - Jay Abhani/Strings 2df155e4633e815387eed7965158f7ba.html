Strings

🔤 What is a String in JavaScript?

A string is a sequence of characters used to represent text. In JavaScript, strings are primitive data types and immutable (cannot be changed once created).

✅ Syntax:

let str1 = "Hello";    // Double quotes
let str2 = 'World';    // Single quotes
let str3 = `Hello World`; // Template literal (backticks)

🧱 How Strings are Stored

Even though strings look like arrays of characters, they are not arrays. You can access characters using bracket notation:

let str = "JavaScript";
console.log(str[0]); // J

But you cannot modify characters directly:

str[0] = "Y"; // ❌ Has no effect
console.log(str); // JavaScript

🔧 Common String Properties

PropertyDescription
lengthReturns the number of characters
let name = "Ujjwal";
console.log(name.length); // 6

🔨 String Methods

JavaScript provides a rich set of built-in string methods. Let's group them:

🏷️ Access and Extraction Methods

MethodDescription
charAt(index)Returns char at given index
charCodeAt(index)Returns Unicode of char at index
slice(start, end)Extracts part of string
substring(start, end)Similar to slice but can't accept negative indexes
substr(start, length)Legacy, use slice instead
let text = "JavaScript";
console.log(text.charAt(0));    // "J"
console.log(text.charCodeAt(0)); // 74
console.log(text.slice(0, 4));  // "Java"
console.log(text.substring(0, 4)); // "Java"
console.log(text.substr(0, 4)); // "Java"

🔍 Search and Match Methods

MethodDescription
indexOf()First occurrence of a substring
lastIndexOf()Last occurrence
includes()Returns true if substring found
startsWith()Checks beginning
endsWith()Checks ending
match()Matches with regex
search()Searches using regex
let sentence = "JavaScript is awesome!";
console.log(sentence.indexOf("is"));      // 11
console.log(sentence.includes("awesome")); // true
console.log(sentence.startsWith("Java")); // true
console.log(sentence.endsWith("!"));      // true

🧼 Modify/Change Methods

MethodDescription
toUpperCase()Converts to uppercase
toLowerCase()Converts to lowercase
trim()Removes whitespace from both ends
trimStart() / trimEnd()Removes leading/trailing spaces
replace()Replaces substring (first match)
replaceAll()Replaces all matches
repeat(n)Repeats string n times
let messy = "   Hello World!   ";
console.log(messy.trim());           // "Hello World!"
console.log(messy.toUpperCase());    // "   HELLO WORLD!   "
console.log(messy.replace("World", "JS")); // "   Hello JS!   "

🔗 Combine and Split Methods

MethodDescription
concat()Joins two or more strings
+ operatorAlso used for string concatenation
split()Splits string into an array
join()(array method) joins array into string
let first = "Hello";
let second = "World";
console.log(first + " " + second); // "Hello World"

let csv = "red,green,blue";
let colors = csv.split(",");
console.log(colors); // ["red", "green", "blue"]

🧠 Template Literals (ES6+)

Use backticks (`) to interpolate variables or expressions directly into strings.

let name = "Ujjwal";
let age = 22;
console.log(`My name is ${name} and I am ${age} years old.`);
// Output: My name is Ujjwal and I am 22 years old.

🔁 Iterating Over Strings

You can use a loop to iterate over string characters.

let word = "Code";
for (let i = 0; i < word.length; i++) {
  console.log(word[i]); // C o d e
}

You can also use for...of:

for (let char of word) {
  console.log(char); // C o d e
}

🛡️ String Immutability

Strings cannot be changed once created. Operations like toUpperCase() or replace() return new strings.

let greeting = "Hello";
let newGreeting = greeting.toUpperCase();
console.log(greeting);     // "Hello"
console.log(newGreeting);  // "HELLO"
