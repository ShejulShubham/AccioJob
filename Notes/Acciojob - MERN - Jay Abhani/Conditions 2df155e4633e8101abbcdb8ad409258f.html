Conditions

✅ What are Conditions in JavaScript?

Conditions are used to perform different actions based on different decisions.

📌 Real-life analogy:

Imagine you're deciding what to wear:

  • If it's raining → wear a raincoat
  • If it's sunny → wear sunglasses
  • Else → wear normal clothes

Similarly, in JavaScript, we use conditions to tell the computer what to do in different situations.

🧠 Common Conditional Statements in JavaScript

StatementDescription
ifExecutes a block if the condition is true
elseExecutes a block if the condition is false
else ifTests a new condition if the first one is false
switchSelects one of many blocks to be executed
Ternary ? :Shorthand for if...else

🔍 1. if Statement

📘 Syntax:

if (condition) {
  // code to run if condition is true
}

✅ Example:

let age = 20;
if (age >= 18) {
  console.log("You are eligible to vote");
}

🔍 2. if...else Statement

📘 Syntax:

if (condition) {
  // true block
} else {
  // false block
}

✅ Example:

let temperature = 15;
if (temperature > 30) {
  console.log("It's hot outside!");
} else {
  console.log("It's cool today!");
}

🔍 3. if...else if...else Ladder

📘 Syntax:

if (condition1) {
  // block 1
} else if (condition2) {
  // block 2
} else {
  // block 3
}

✅ Example:

let score = 75;

if (score >= 90) {
  console.log("Grade A");
} else if (score >= 70) {
  console.log("Grade B");
} else if (score >= 50) {
  console.log("Grade C");
} else {
  console.log("Fail");
}

🔍 4. Ternary Operator (? :)

It is a shorthand way of writing if...else.

📘 Syntax:

condition ? value_if_true : value_if_false;

✅ Example:

let isLoggedIn = true;
let message = isLoggedIn ? "Welcome back!" : "Please log in.";
console.log(message);

🔍 5. switch Statement

Useful when you have many exact match cases.

📘 Syntax:

switch(expression) {
  case value1:
    // code
    break;
  case value2:
    // code
    break;
  default:
    // code
}

✅ Example:

let day = "Wednesday";

switch (day) {
  case "Monday":
    console.log("Start of the week");
    break;
  case "Wednesday":
    console.log("Midweek day");
    break;
  case "Friday":
    console.log("Almost weekend!");
    break;
  default:
    console.log("Just a regular day");
}

🧪 Logical Operators in Conditions

You can use:

  • && (AND)
  • || (OR)
  • ! (NOT)

✅ Example:

let isStudent = true;
let age = 19;

if (isStudent && age < 25) {
  console.log("Eligible for student discount");
}

🧠 More Practical Examples

✅ Check Even or Odd:

let num = 7;

if (num % 2 === 0) {
  console.log("Even");
} else {
  console.log("Odd");
}

✅ Login Status Check:

let username = "admin";
let password = "1234";

if (username === "admin" && password === "1234") {
  console.log("Login successful");
} else {
  console.log("Invalid credentials");
}

✅ Traffic Light System:

let signal = "red";

if (signal === "green") {
  console.log("Go");
} else if (signal === "yellow") {
  console.log("Get ready");
} else if (signal === "red") {
  console.log("Stop");
} else {
  console.log("Invalid signal");
}
