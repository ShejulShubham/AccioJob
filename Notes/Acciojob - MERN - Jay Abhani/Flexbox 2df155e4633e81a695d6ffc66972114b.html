Assignment - 1

🧩 Assignment 1 — Pricing Section (Flexbox)

✅ What you will build

Create a row of three pricing cards: Starter, Pro, Enterprise.

  • The Pro card should be visually wider (grow ratio 1 : 2 : 1).
  • All cards should look equal height regardless of content.
  • The CTA button stays at the bottom of each card (use inner flex column).
  • Use gaps between cards and padding around the container.
  • Wrapping is allowed (on small widths, cards may move to the next line).
  • Do NOT use media queries.

Solution:

<style>
  .pricing {
    display: flex;
    flex-wrap: wrap;            /* allow wrapping */
    gap: 16px;                  /* spacing between cards */
    padding: 16px;              /* outer padding */
    background: #f6f8fb;        /* subtle section styling */
    border: 1px solid #e6ebf2;
  }

  .card {
    display: flex;              /* inner column layout */
    flex-direction: column;
    flex: 1 1 280px;            /* default flexible card */
    border: 1px solid #ddd;
    background: #fff;
    padding: 16px;
    border-radius: 12px;
    box-shadow: 0 2px 6px rgba(0,0,0,.06);
  }

  .card h3 {
    margin: 0 0 8px;
    font-size: 20px;
  }
  .price {
    font-size: 28px;
    font-weight: 700;
    margin-bottom: 8px;
  }
  .features {
    display: flex;              /* vertical stack with gap */
    flex-direction: column;
    gap: 6px;
    list-style: none;
    padding: 0;
    margin: 0 0 16px;
  }

  .fill { flex: 1; }           /* pushes the button to the bottom */

  .card button {
    padding: 10px 12px;
    border: none;
    border-radius: 8px;
    background: #0b74e5;
    color: #fff;
    font-weight: 600;
    cursor: pointer;
  }

  /* grow ratios: 1 : 2 : 1 */
  .starter    { flex: 1 1 280px; }
  .pro        { flex: 2 1 320px; }  /* wider middle card */
  .enterprise { flex: 1 1 280px; }
</style>
