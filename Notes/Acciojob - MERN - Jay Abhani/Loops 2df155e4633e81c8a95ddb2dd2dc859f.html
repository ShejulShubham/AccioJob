Loops

🔁 1. for loop

✅ Syntax:

for (initialization; condition; increment/decrement) {
  // code block to execute
}

✅ Explanation:

  • Initialization: runs once before the loop starts.
  • Condition: checked before every loop iteration. If true, the loop runs.
  • Increment/Decrement: updates the loop variable after each iteration.

✅ Example 1: Print numbers from 1 to 5

for (let i = 1; i <= 5; i++) {
  console.log(i);
}
// Output: 1 2 3 4 5

✅ Example 2: Print even numbers between 1 and 10

for (let i = 2; i <= 10; i += 2) {
  console.log(i);
}
// Output: 2 4 6 8 10

🔁 2. while loop

✅ Syntax:

while (condition) {
  // code block to execute
}

✅ Explanation:

  • The condition is checked before each loop iteration.
  • If the condition is true, the loop body runs.
  • You must manually update the loop variable, or it may run infinitely.

✅ Example 1: Print numbers from 1 to 5

let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
// Output: 1 2 3 4 5

✅ Example 2: Count down from 5 to 1

let count = 5;
while (count >= 1) {
  console.log(count);
  count--;
}
// Output: 5 4 3 2 1

🔁 3. do...while loop

✅ Syntax:

do {
  // code block to execute
} while (condition);

✅ Explanation:

  • The code block runs first, and then the condition is checked.
  • So the loop will run at least once, even if the condition is false initially.

✅ Example 1: Print numbers from 1 to 5

let i = 1;
do {
  console.log(i);
  i++;
} while (i <= 5);
// Output: 1 2 3 4 5

✅ Example 2: Condition false at start

let num = 10;
do {
  console.log("This will run once even if condition is false.");
} while (num < 5);
// Output: This will run once even if condition is false.

📌 Comparison Summary

Loop TypeWhen condition is checkedRuns at least once?

🔄 Practical Use Cases

💡 Loop through an array

const fruits = ["apple", "banana", "orange"];

for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}
// Output: apple banana orange

💡 Print first 10 multiples of 3 using while

let i = 1;
while (i <= 10) {
  console.log(3 * i);
  i++;
}
// Output: 3 6 9 12 15 18 21 24 27 30

✅ Conclusion:

  • Use for loop when the number of iterations is known.
  • Use while loop when the number of iterations is not known and depends on some condition.
  • Use do...while when you want the loop to run at least once no matter what.
