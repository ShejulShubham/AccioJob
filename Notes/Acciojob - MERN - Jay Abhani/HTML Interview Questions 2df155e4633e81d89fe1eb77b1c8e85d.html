HTML Interview Questions

✅ Basic HTML Interview Questions & Answers

1. What is HTML?

Answer:

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It uses tags to define elements like headings, paragraphs, links, images, etc.

2. What are HTML tags?

Answer:

HTML tags are the building blocks of HTML. They define how elements should be displayed in the browser. Most tags come in pairs like <p>...</p> — an opening tag and a closing tag.

3. What is the difference between HTML and HTML5?

Answer:

  • HTML is the older version, while HTML5 is the latest standard.
  • HTML5 supports multimedia (audio/video) and new elements like <header>, <footer>, <article>, etc.
  • HTML5 has better support for mobile and responsive design.

4. What is the structure of a basic HTML document?

Answer:

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

5. What is the purpose of the <!DOCTYPE> declaration?

Answer:

It tells the browser which version of HTML is being used so that it renders the page correctly.

6. What are semantic HTML elements?

Answer:

Semantic elements clearly describe their meaning. Examples:

  • <header>: Header of a section/page
  • <nav>: Navigation links
  • <article>: Independent content
  • <footer>: Footer of a section/page

7. What is the difference between <div> and <span>?

Answer:

  • <div> is a block-level element (takes full width).
  • <span> is an inline element (takes only as much width as needed).

8. How do you insert an image in HTML?

Answer:

<img src="image.jpg" alt="Description of image" />
  • src specifies the image path.
  • alt provides alternate text if the image can't be loaded.

9. How do you create a hyperlink in HTML?

Answer:

<a href="https://example.com">Visit Example</a>
  • href specifies the destination URL.

10. What is the use of the <meta> tag?

Answer:

It provides metadata about the HTML document like character encoding, author, viewport settings, etc.

Example:

<meta charset="UTF-8" />
<meta name="viewport" content="width=device-width, initial-scale=1.0">

🧩 Intermediate HTML Questions for Freshers

11. What is the difference between id and class?

Answer:

  • id: Unique identifier (used once per page).
  • class: Can be used for multiple elements.

12. Can you nest one HTML element inside another?

Answer:

Yes, nesting is common in HTML. Example:

<p>This is a <strong>bold</strong> word.</p>

13. What is the purpose of the <form> tag?

Answer:

It is used to create a form for user input, which can be submitted to a server.

14. What are self-closing tags?

Answer:

Tags that don’t require a closing tag.

Examples: <img />, <br />, <input />

15. How is HTML different from CSS and JavaScript?

Answer:

  • HTML: Structure
  • CSS: Style
  • JavaScript: Behavior/Interactivity

🧠 Practice-Based & Practical Questions

16. How do you make a list in HTML?

Answer:

  • Ordered List (numbered):
<ol>
  <li>First</li>
  <li>Second</li>
</ol>
  • Unordered List (bullets):
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

17. How do you create a table in HTML?

Answer:

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>22</td>
  </tr>
</table>

18. What is the alt attribute in the <img> tag used for?

Answer:

It provides descriptive text for screen readers and is shown if the image fails to load.

19. What is a block vs inline element?

Answer:

  • Block elements: Take full width. (e.g., <div>, <p>, <h1>)
  • Inline elements: Fit within content. (e.g., <a>, <span>, <strong>)

20. How to add comments in HTML?

Answer:

<!-- This is a comment -->
