📘 JavaScript Operators - Full Guide

🔹 What are Operators?

Operators are symbols or keywords used to perform operations on variables and values. For example: +, -, *, =, etc.

JavaScript has different types of operators, which are grouped based on their purpose.

🔸 8. Ternary Operator

Shorthand for if-else

let age = 18;
let result = (age >= 18) ? "Adult" : "Minor";
console.log(result); // Adult

🔸 9. Nullish Coalescing (??)

Returns right-hand value only if left is null or undefined.

let username = null;
let name = username ?? "Guest";
console.log(name); // Guest

🔸 10. Optional Chaining (?.)

Safely access nested object properties without throwing an error.

let user = {
  name: "Jay",
  address: {
    city: "Ahmedabad"
  }
};

console.log(user.address?.city); // Ahmedabad
console.log(user.profile?.age);  // undefined (no error)
