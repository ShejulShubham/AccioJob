Spread and Rest Operators

🔹 ... Spread vs Rest Operator in JavaScript

FeatureSpread OperatorRest Operator
PurposeExpands/unpacks valuesGathers/packs values
Use CaseIn function calls, arrays, or objectsIn function parameters or destructuring
Syntax PositionUsed on the right side (expansion)Used on the left side (collection)

✅ Spread Operator (...)

Used to spread out elements of an array or properties of an object.

🔸 Spread with Arrays

let arr1 = [1, 2];
let arr2 = [3, 4];
let combined = [...arr1, ...arr2];
console.log(combined); // [1, 2, 3, 4]

🔸 Copying an Array

let original = [1, 2, 3];
let copy = [...original];

🔸 Spread with Strings

let str = "hello";
let chars = [...str];
console.log(chars); // ['h', 'e', 'l', 'l', 'o']

🔸 Spread with Objects

let obj1 = { a: 1, b: 2 };
let obj2 = { b: 3, c: 4 };
let merged = { ...obj1, ...obj2 };
console.log(merged); // { a: 1, b: 3, c: 4 }
✅ Later properties override earlier ones in object spread.

✅ Rest Operator (...)

Used to collect remaining elements into an array or object.

🔸 Rest in Function Parameters

function sum(...numbers) {
  return numbers.reduce((a, b) => a + b);
}

console.log(sum(1, 2, 3)); // 6

🔸 Rest in Array Destructuring

let [first, ...rest] = [1, 2, 3, 4];
console.log(first); // 1
console.log(rest);  // [2, 3, 4]

🔸 Rest in Object Destructuring

let user = { name: "Alice", age: 25, city: "Mumbai" };
let { name, ...rest } = user;
console.log(name); // Alice
console.log(rest); // { age: 25, city: "Mumbai" }

✨ Example Combining Both:

function greet(greeting, ...names) {
  console.log(`${greeting}, ${names.join(" & ")}`);
}

let people = ["Alice", "Bob"];
greet("Hello", ...people); // Hello, Alice & Bob
