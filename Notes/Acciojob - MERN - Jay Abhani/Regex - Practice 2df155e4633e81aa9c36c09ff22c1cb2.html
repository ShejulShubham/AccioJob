2. What is the purpose of the g flag in regular expressions?

✅ Answer:

The g flag stands for global, and it is used to match all occurrences of the pattern in the string instead of stopping after the first match.

💡 Example:

let text = "hello hello";
let pattern = /hello/g;
console.log(text.match(pattern));  // ["hello", "hello"]

3. Which method is used to replace text using regex?

✅ Answer:

replace() method.

💡 Example:

let text = "hello world";
console.log(text.replace(/hello/, "hi"));  // "hi world"

4. What does the regex /\d/ match?

✅ Answer:

It matches any single digit from 0 to 9.

💡 Example:

let text = "abc123";
console.log(/\d/.test(text));  // true

5. What is the output of this code?

💡 Explanation:

The split() method splits the string using the regex pattern /,/ which matches commas. So, the string is split at each comma into an array.

✅ Problem 1: Basic Email Validation

Task:

Write a regex to validate a simple email that follows the format:

username@domain.com
function isValidEmail(email) {
  let pattern = /^\w+@\w+\.\w+$/;
  return pattern.test(email);
}

console.log(isValidEmail("john@example.com")); // true
console.log(isValidEmail("john@.com"));        // false

Explanation:

  • ^ → start of string
  • \w+ → one or more word characters (username)
  • @ → literal at symbol
  • \w+ → one or more word characters (domain)
  • \. → dot
  • \w+ → one or more word characters (TLD like com/net/org)
  • $ → end of string

✅ Problem 3: Basic Password Validation

Task:

Check if a password is at least 6 characters long and contains at least one digit.

function isValidPassword(pwd) {
  let pattern = /^(?=.*\d).{6,}$/;
  return pattern.test(pwd);
}

console.log(isValidPassword("abc123"));   // true
console.log(isValidPassword("abcdef"));   // false (no digit)

Explanation:

  • (?=.*\d) → positive lookahead to ensure at least one digit
  • .{6,} → string should be at least 6 characters long
