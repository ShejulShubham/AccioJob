DOM Basics Exercises

🧪 Exercise 1: Highlight Active Navigation Tab

🎯 Task:

You are building a navbar for a personal portfolio website. When the page loads, the "Home" tab should appear highlighted to indicate that it is the active page.

🧾 Instructions:

  • Select the <li> element with id="home-tab" using a DOM selector.
  • Add a CSS class called "active-tab" to that element using classList.add().

This class will be used to visually highlight the active tab.

🧱 Initial HTML Code:

<ul id="navbar">
  <li id="home-tab">Home</li>
  <li>About</li>
  <li>Projects</li>
  <li>Contact</li>
</ul>

🎨 CSS for Styling (Just for Visualization):

.active-tab {
  font-weight: bold;
  color: royalblue;
  text-decoration: underline;
}

🧪 Exercise 2: Add Testimonial Section Dynamically

🎯 Task:

You're building a landing page for a product. The testimonial section should be created and added dynamically using JavaScript when the page loads.

🧾 Instructions:

  • Use createElement() to create a new <p> element.
  • Use textContent to add this text:

    "This product changed my life!"

  • Use appendChild() to add this <p> inside the existing <div id="testimonials">.

🧱 Initial HTML Code:

<div id="testimonials">
  <h2>What Our Users Say</h2>
</div>

🧪 Exercise 3: Update Hero Section Title and Style on Button Click

🎯 Task:

You're working on the hero section of a portfolio homepage. When a user clicks a button, the main title should update and get visually highlighted.

🧾 Instructions:

  • Use getElementById() or querySelector() to select the <h1 id="hero-title">.
  • When the user clicks the button:
    • Change the text to "Welcome to My Portfolio" using textContent.
    • Add a class "highlight" to the <h1> using classList.add().

🧱 Initial HTML Code:

<section id="hero">
  <h1 id="hero-title">Hello World!</h1>
  <button id="update-btn">Update Title</button>
</section>

🎨 Optional CSS for .highlight:

.highlight {
  color: #007bff;
  font-size: 2.5rem;
  transition: 0.3s ease;
}

🧪 Exercise 4: Add and Remove Newsletter Pop-up

🎯 Task:

On your blog website, a newsletter pop-up should appear when the user clicks the “Show Newsletter” button, and disappear when they click “Close”.

🧾 Instructions:

  • When the "Show Newsletter" button is clicked:
    • Use createElement() to create a <div> with text: "Subscribe to our Newsletter".
    • Give it an id="newsletter" and add it to <body> using appendChild().
  • When the "Close" button is clicked:
    • Use getElementById() and removeChild() to remove the newsletter popup from the DOM.

🧱 Initial HTML Code:

<button id="show-btn">Show Newsletter</button>
<button id="close-btn">Close</button>

🧪 Exercise 5: Toggle Dark Theme on Button Click

🎯 Task:

You’re building a dashboard or portfolio site that supports light and dark themes. When the user clicks the “Toggle Theme” button, the dark-theme class should be toggled on the <body> element.

🧾 Instructions:

  • Select the <body> element.
  • When the button is clicked:
    • Use classList.toggle() to add or remove the "dark-theme" class on <body>.

🧱 Initial HTML Code:

<button id="theme-toggle-btn">Toggle Theme</button>

🎨 Optional CSS for .dark-theme:

.dark-theme {
  background-color: #121212;
  color: #ffffff;
}
