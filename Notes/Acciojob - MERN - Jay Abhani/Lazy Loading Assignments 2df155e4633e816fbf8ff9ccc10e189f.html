🧪 PRACTICE ASSIGNMENT 6

Multiple Lazy Pages + Default Route

🎯 Objective

Practice multiple lazy routes with a default landing page.

📝 Problem Statement

Create a React app with:

  • / → Landing page
  • /about → About page
  • /contact → Contact page

All pages must be lazy loaded.

📋 Instructions

  1. Create a pages/ folder.
  1. Lazy load all pages.
  1. Wrap <Routes> with a single Suspense.
  1. Add navigation links.
  1. Confirm each page logs when loaded.

🚫 Constraints

❌ Do NOT lazy load navigation

❌ Do NOT create multiple Suspense wrappers

📁 Folder Structure

src/
 ├── pages/
 │    ├── Landing.jsx
 │    ├── About.jsx
 │    └── Contact.jsx
 ├── App.jsx

✅ COMPLETE SOLUTION

pages/Landing.jsx

export default function Landing() {
  console.log("Landing loaded");
  return <h1>Landing Page</h1>;
}

pages/About.jsx

export default function About() {
  console.log("About loaded");
  return <h1>About Page</h1>;
}

pages/Contact.jsx

export default function Contact() {
  console.log("Contact loaded");
  return <h1>Contact Page</h1>;
}

App.jsx

import React, { Suspense } from "react";
import { Routes, Route, Link } from "react-router-dom";

const Landing = React.lazy(() => import("./pages/Landing"));
const About = React.lazy(() => import("./pages/About"));
const Contact = React.lazy(() => import("./pages/Contact"));

function App() {
  return (
    <>
      <nav>
        <Link to="/">Home</Link> |{" "}
        <Link to="/about">About</Link> |{" "}
        <Link to="/contact">Contact</Link>
      </nav>

      <Suspense fallback={<h2>Loading page...</h2>}>
        <Routes>
          <Route path="/" element={<Landing />} />
          <Route path="/about" element={<About />} />
          <Route path="/contact" element={<Contact />} />
        </Routes>
      </Suspense>
    </>
  );
}

export default App;
